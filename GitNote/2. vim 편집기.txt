1. vim 
    1) 대표적인 리눅스의 기본 편집기 
    2) vim에서 문서 작성, 저장하기 
    3) vim에는 문서를 작성하는 '입력 모드'와
               문서를 저장하는 'ex모드'가 있음.

              
                         i 누름
           ex 모드  ------------------->  입력 모드 (INSERT)
         (저장,종료) <------------------  (텍스트 입력,수정)
                         esc 누름   

2. vim의 ex 모드 명령
    1) 콜론(:)으로 시작함 
    2) :w 또는 :write               저장 
       :q 또는 :quit                종료 
       :wq                          편집하던 문서 저장하고 종료 
       :q!                          편집하던 문서 저장하지 않고 종료 
                                    확장자 .swp인 임시파일 생김.
       :wq 파일명                    편집하던 문서를 지정한 파일이름으로 저장                                                                    

3. cat 
    1) concatenate(연결시키다) 줄임말 
    2) 텍스트 문서 확인하기 
    3) 명령
        cat file            file의 내용을 화면에 표시함 
        cat file > newfile  file을 차례로 연결해서 새로운 파일인 newfile을 만듦
