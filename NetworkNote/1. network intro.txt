1. 네트워크(Network)
    1) 컴퓨터와 컴퓨터를 연결한 것
    2) 사람들끼리 촘촘히 얽혀서 일을 할수 있는 환경
    3) 컴퓨터, 스마트폰와 같은 여러 장치들이 서로 유기적으로 연결되어
       있는 환경을 제공하는 것 
    4) 각종 통신 장비들이 서로 그물망처럼 연결되어 데이터를 교환하거나
       협력하여 일할 수 있게 해주는 통신망 

2. 랜 (Local Area Network, LAN)
    1) 특정 건물 안이나 특정 지역을 범위로 하는 네트워크 
    2) 근거리 통신망
    3) 3가지 방식 연결
        - 스타(star)형
            - 하나의 허브(Hub)에 여러 대의 컴퓨터를 연결하는 방식 
        - 링(ring)형  
            - 개별 컴퓨터가 서로 원처럼 연결되어 있는 구성 방식 
        - 버스(bus)형
            - 하나의 긴 선(케이블)에 컴퓨터를 포함한 모든 주변 장치를
              연결하는 방식                  
            - 일번적으로는 버스형을 많이 사용         

3. 왠(Wide Area Network, WAN)
    1) 2개 이상의 랜을 연결한 것  
    2) 광역 통신망                          

4. 네트워크 구성 장치
    1) 컴퓨터끼리 통신하기 위해서는 케이블을 이용해서 서로 연결해야 함.
    2) 허브, 스위치(Switch), 라우터(Router), 브리지(Bridge)
    3) 대역폭 (bandwidth)
        - 1초당 처리할 수 있는 데이터의 양
        - 원통이 넓으면 넓을수록 데이터를 더 많이 처리할 수 있음 
        - bps(bit per second)
    4) 비트와 바이트 
        - 데이터 처리 단위 
        - 네트워크 통신에 있어서 가장 기본적인 단위 
        - 바이트를 1024(= 2^10)배 하면 킬로바이트(Kilobyte)가 되고,
          킬로바이트의 1024배를 하면 메가바이트(Megabyte)가 되고,
          메가바이트의 1024배를 하면 기가바이트(Gigabyte)가 되며,
          기가바이트에 1024배를 하면 테라바이트(Terabyte)가 됨
