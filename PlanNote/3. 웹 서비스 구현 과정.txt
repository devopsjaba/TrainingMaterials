1. 6단계
    1) 서비스 구상 단계
        - 사업 계획 
        - 서비스 기획 

    2) 서비스 구현 단계 (오류가 발생한 단계로 이동)
        - 서비스 분석 (요구 사항 분석)
        - 화면 설계 
        - 디자인 
        - 퍼블리싱 
        - 개발 
        - 테스트 

2. 준비 단계
    1) 주요 메뉴만 추려 웹 사이트 맵 구성함 
        - 웹 사이트 메뉴 한눈에 볼 수 있는 조감도 

    2) '기능'의 의미 
        - 기능은 일반적으로 '특정한 동작'을 뜻함 
        - 프로세스 (process)
            - 예) 회원 가입 기능, 결제 기능, 게시판 기능 
        - 엘리먼트 (Element)                
            - 예) 화면을 구성하는 여러 요소 가운데 특정 기능을 수행하는 것 
                - 에디터 기능, 옵션 기능, 등록 기능
