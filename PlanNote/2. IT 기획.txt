1. 서비스 기획자 - 어떤 서비스를 만들까?
    1) 창의성 있는 아이디어 필요 
    2) 다른 업무 담당자와 협업
    3) 무조건 성공하는 서비스를 만들어야 함 
    4) 서비스 기획자의 대표 산출물은 '서비스 기획서'임      

    5) 세상에 필요한 서비스 생각하기 
    6) 서비스 정의하기 
    7) 수익 모델 설계하기 
    8) 이용 프로세스 설계하기 
    9) 프로토타입(prototype) 설계하기 

2. 구축 기획자 -  원하는 웹 사이트를 구축한다!
    1) SI (system integration)
        - 의뢰자가 원하는 웹 사이트나 시스템을 제작해 주는 것 
        - SI 기획자 (구축 기획자)
    2) 서비스의 시작부터 끝까지 모든 프로세스와 화면을 설계하는 업무 
        - 서비스 분석 단계 (요구사항 분석하기)
            - 정보 구조 (IA, Information Architecture)         
            - 기능 정의서 
    3) 화면 설계 단계 (이용 프로세스 정리하기)
        - 이용 flow
        - 화면 정의서 작성하기        
    4) 디자인, 퍼블리싱 단계 (웹 디자인, 웹 퍼블리싱 검토하기)   
    5) 개발, 테스트 단계           

3. 운영 기획자 (웹 사이트를 더 좋게 만듦)
    1) 이벤트 기획하기 
    2) 마케팅 지원하기
