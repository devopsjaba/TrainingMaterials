1. 프로그래밍 (Programming)
    1) 컴퓨터 프로그램 만드는 일 
    2) 컴퓨터가 일을 하도록 컴퓨터 언어로 명령을 만들고 
       컴퓨터로 하여금 그 명령을 실행하게 하는 것 
    3) 프로그램   
        - 컴퓨터에게 일을 시키는 명령의 집합 
        - 프로그래밍 언어 문법에 잘 맞는지 확인하고,
          컴퓨터가 이해할 수 있는 언어로 번역(컴파일)해야 함 
        - 프로그램에 들어 있는 명령(소스 코드)집합과 
          컴파일된 결과물까지 포함함 

2. 프로그래밍 언어
    1) 고급 언어
        - 인간이 이해하기 쉬운 문장을 사용해 컴퓨터가 수행해야 할 
          작업을 정의한 언어 
        - C, C++, Java, Python....

    2) 어셈블리어 
        - 기계어와 1대 1로 대응되는 언어 

    3) 기계어  
        - 2진 데이터 (001011001...)로 구성되 있어 컴퓨터가 직접
          알아들을 수 있는 언어

                              런타임 라이브러리 
                            - 자바 가상 머신 (JVM)

6. 자바 프로젝트 
    1) 1단계
        - 자바 프로젝트 생성
        - 프로젝트 폴더 생성      

       2단계
        - 패키지 생성하기 
        - 하위 폴더 생성

       3단계 
        - 자바 소스 생성하기 
        - 소스 파일 생성 (.java)

       4단계
        - 컴파일 및 바이트 코드 생성하기 (.class)

       5단계
        - 실행하기
    
    2) 프로젝트명 : HelloWorld
       패키지명 : kr.co.kosta,
                 디폴트 패키지 (패키지를 지정하지 않음)     
       소스 파일명 : Hello.java,
                    Main.java 

    3) 자바 소스 코드의 실행 과정 
        - .java 소스 파일 생성
        - 파일 저장 시 자동 컴파일  
        - .class 바이트 코드 파일 생성    
        - 실행 (Ctrl + F11)
        - JVM은 메모리 할당 (메소드 영역, 스택 영역, 힙 영역)
        - 메소드 영역에 클래스 로딩    
        - main() 메서드 실행             

7. 콘솔 출력 메서드와 문자열 출력 
    1) 문자열 표현하기 
        - 문자열은 String 자료형으로 저장됨 
        - 값의 표현은 "안녕", "Hello"등과 같이 반드시 큰따옴표("")안에 표기 
    2) 줄 바꾸면서 출력하기  
    3) 변수에 넣어 출력하기 
    4) 1줄로 출력하기 - System.out.print()
    5) 형식대로 출력하기 - System.out.printf()
        - System.out.printf("출력 포맷", 인자, 인자,...)
            - 출력 포맷
                - %d, %o, %x, %f, %s
