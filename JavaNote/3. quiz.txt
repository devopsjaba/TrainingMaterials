   자바 개발 도구(JDK), 자바 개발 환경(JRE)의 포함 관계를 작성하시오.
    - JDK > JRE > JVM 

2. 이클립스 단축키 
        동작                        단축키 
    - 자동 임포트                   Ctrl + Shift + O 
    - 1줄 주석 설정 및 해제          Ctrl + /     
    - 자동 정렬                     Ctrl + Shift + F 
    - 폰트 확대                     Ctrl + +

    - 자바 가상 머신을 사용해 하나의 소스 코드를 컴파일한 결과로,
      어떤 플랫폼에서든 실행 할 수 있음.
