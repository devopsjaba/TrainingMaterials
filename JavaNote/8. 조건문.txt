1. 조건문이란?
        만약(나이가 19살 이상이라면) {
            편의점에서 주류를 구매합니다.
        }

    1) if 단일 구문 
        if (조건식) {
            수행문;     // 조건식이 참일 경우에 이 문장을 수행 
        }

    2) if-else 구문 
        if (조건식) {
            수행문;     // 조건식이 true일때만 실행
        } 
        else {
            수행문;     // 조건식이 false일때만 실행                
        }   

    3) if-else if-else 구문 
        if(조건식 1) {
            수행문;       // 조건식1이 true일때만 실행 제어문 탈출 
        }
        else if(조건식2) {
            수행문;      // 조건식2가 true일때만 실행 제어문 탈출   
        }
        ...
        else if(조건식N) {
            수행문;      // 조건식N이 true일때만 실행 제어문 탈출 
        }
        else {
            수행문;      // 조건식이 false일때만 실행 제어문 탈출  
        }

    4) switch-case문 
        - switch(점프 위치 변수) {          // 정수, 문자, 문자열 사용 가능
            case 위치값 1:
                실행 구문;
            case 위치값 2:
                실행 구문;
                ...
            case 위치값 n:
                실행 구문;
            default:               // 일치하는 값이 없을 때 위치로 이동(생략가능)
                실행 구문;                
          }        

    5) java14부터 지원되는 switch-case문          
        - 간단하게 쉼표(,)로 조건 구문 
        - 식으로 표현하여 반환 값을 받을 수 있음. 리턴 값이 없을 경우는 오류가 생김
        - yield 키워드 사용
