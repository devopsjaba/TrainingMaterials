1. 컴퓨터 언어
    1) 우리가 언어를 이해하는 방법 
    2) 컴퓨터가 대화를 이해하는 방법 
        - 컴퓨터는 CPU에서 비트(bit)라는 단위로 생각을 함
            - 비트의 의미 
                - 조금, 약간, 잠깐 
                - 한 조각 
            - 비트에는 딱 두 가지 뜻만 있음
                - 전등의 스위치를 켜고 끄는 것
                - 켜다(on), 끄다(off)
                  예        아니오
                  참        거짓
                  1         0 
                - 신호(Signal)로 이해.
            - 비트는 0 또는 1 중 하나의 정보를 가지는 쪼그만 정보 단위. 

2. 코드 (Code)        
    1) 0과 1의 조합의 약속  
    2) 각기 다른 비트의 조합이 의미하는 것들
    3) 코드의 사전적 의미
        - 암호, 부호 
        - 관례, 규칙
        - 법규, 규정 

3. 바이트 
    1) 비트 8개를 묶어서 1바이트(Byte)라는 새로운 단위를 만듦
    2) 영어를 모두 표현할 수 있는 최소한의 크기  
    3) 사전적 의미
        - 한 입(베어 문 조각)     
    4) 8비트, 즉 1 바이트로 표현된 약속 
    5) 예

        아니요  예 예 아니요 아니요 아니요 아니요 예    
          0    1   1      0     0     0      0    1  ==> 영어 소문자 a 

        아니요 아니요 예 아니요 아니요 아니요 아니요 예    
          0    1   0      0     0     0      0    1  ==> 영어 대문자 A 

    6) 사람                CPU
                아스키            
        Hello  =======> 01100011 
        Hi     <======  11001100

4. 인코딩, 디코딩  
    1) 인코딩(Encoding)                        
        - 사람의 언어를 컴퓨터 언어로 바꾸는 과정 
    2) 디코딩(Decoding)
        - 그 반대의 과정         

5. 아스키, 유니코드
    1) 2비트로 할수 있는 약속 
        - 예-예, 예-아니오, 아니요-예, 아니요-아니요 
            ==> 4가지 약속을 정할수 있음
    2) 3비트로 할수 있는 약속
        - 예-예-예, 예-예-아니오, 예-아니요-예, 예-아니요-아니요  
        - 아니오-예-예, 아니오-예-아니오, 아니오-아니요-예, 아니오-아니요-아니요       
            ==> 8가지 약속을 정할수 있음
    3) 1비트가 늘어나면 약속이 가능한 조합의 가짓수는 2배씩 늘어남 
        - 8비트는 2^8 (2를 8번 곱한) 256가지를 약속할수 있음 
            - 8비트 단위로 약속을 정한 아스키(ASCII)
        - 16비트는 2^16 (2를 16번 곱한) 65536가지를 약속할수 있음
            - 16비트 단위로 약속을 정한 유니코드(UNICODE)    

6. 아스키 (ASCII)
    1) 알파벳을 위한 약속 (컴퓨터가 이해하는 숫자)
    2) American Standard Code for Information Interchange 
    3) 정보 교환을 위한 미국 표준 약속 
    4) 영어 대문자, 영어 소문자, 숫자, 특수문자를 8비트안에 약속함                    

7. 유니코드 (UNICODE)
    1) 모든 언어를 위한 약속 
    2) 최소 단위인 8비트(1바이트)의 두배 크기로 늘려서 16비트(2바이트)를
       기본 공간으로 정함 
    3) 기존에 쓰던 아스키 코드를 유니코드 안에 집어놓음 
        - 영문자는 아스키 코드의 8비트 앞에 8비트 공간을 추가하고,
          추가된 공간에 0을 여덟 번 넣어서 만듦. 
        - UTF-8
        - Universal Coded Character Set + Transformation Format 
            
8. 기계어 (Machine code)
    1) 0과 1의 비트 조합
    2) 기계가 가장 좋아하는 언어 
    3) 기계마다 다른 기계어 

9. 컴파일 (Compile) 
    1) 사람이 알아볼 수 있는 단어로 작성된 언어를 
       컴퓨터가 이해하는 언어로 변환하는 작업    
    2) 사전적 의미 
        - 엮다, 편집하다, 편찬하다   
    3) 컴파일러(Compiler)
        - 편집하는 도구 
        - 컴퓨터 언어 번역기 
        - 예

            프로그래머           자바 컴파일러              컴퓨터
             int x, y;                                      
             x = 10 + 2;
             y = x + 4;
