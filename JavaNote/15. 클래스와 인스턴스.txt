1. 클래스 사용과 main() 메서드 
    1) 프로그램을 시작하는 main() 메서드 
    2) new 예약어로 클래스 생성하기 
        - 클래스형 (참조)변수 = new 생성자;

                정의                    생성(인스턴스화)
        - 학생 --------> Student 클래스  ----------->    student1 인스턴스 
                                        ----------->    student2 인스턴스
                                        ----------->    student3 인스턴스 

    3) 참조 변수 사용하기 
        - 포인트 연산자 사용
            - 참조 변수.멤버 변수
            - 참조 변수.메서드명() 

    4) 인스턴스와 힙(heap) 메모리 
        - 예)
            Student student1 = new Student();
                               ----
                               힙 메모리에 저장
        - 
            스택(stack)                   힙(heap)
               student1   -------------> Student 클래스 생성
        - 힙(heap)
            - 프로그램에서 사용하는 동적 메모리(dynamic memory) 공간임 
            - 객체가 생성될 때 사용하는 공간임 
            - 힙은 동적으로 할당되며 끝나면 자동으로 가비지 컬렉터(garbage collector)가
              자동으로 메모리를 해제해 줌.

2. 용어 정리
    1) 객체 : 객체 지향 프로그래밍의 대상, 생성된 인스턴스 
       클래스 : 객체를 프로그래밍하기 위해 코드로 만든 상태
    2) 인스턴스 : 클래스가 메모리에 생성된 상태   
    3) 멤버 변수 : 클래스의 속성, 특성 
       메서드 : 멤버 변수를 이용하여 클래스의 기능을 구현 
    4) 참조 변수 : 메모리에 생성된 인스턴스를 가리키는 변수 
       참조 값 : 생성된 인스턴스의 메모리 주소 값
