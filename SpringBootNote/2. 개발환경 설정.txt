2. Open JDK vs Oracle JDK 
    1) JDK 
        - 자바 기반의 개발환경에서 가장 기본이 되는 소프트웨어
        - JDK를 설치하여 JVM(Java Virtual Machine)을 사용 
        -    JVM  
              OS 
    2) 히스토리 
        - Sun Microsystems
            - 2006년에 Java를 GNU GPL 라이센스로 등록함 
        - JDK 
            - 무료 버전의 OpenJDK
            - 유료 버전의 OracleJDK     
        - 2018년 6월까지 BCL 라이선스 정책 적용
            - 개인적으로 사용하는 경우 무료 
            - 상업적 목적으로 하는 경우 요금 지불 
        - 2018년 7월부터 구독형 라이센스로 전환   
        - 초창기 OpenJDK는 성능이나 안정성에 다소 문제가 있었음
          그러나 Java 11 부터는 OracleJDK와 거의 동일해짐
