          const 변수명

    2) 비교            선언하지 않고 사용할 경우        재선언      재할당

2. 자료형 
    1) typeof(값 or 변수)
    2) 숫자형          
    3) 문자열 
        - 템플릿 리터럴 
            - 백팃(`)을 사용해 문자열을 만듦 
    4) 논리형       
    5) undefined
        - 값이 할당되지 않았을 때 변수의 초기값
        - 값이면서 동시에 자료형 
        - 변수를 만들기만 하고 값을 할당하지 않았을 때
          자바스크립트에서 자동으로 부여하는 초깃값 
       null 
        - 값이 없거나 유효하지 않은 값이라는 의미임. 

    6) 객체
        - 여러 개의 원시 타입을 하나로 묶어 놓은 것
        - 배열, 함수도 객체로 취급함 
        - 객체 정의하기 
            - 중괄호({}) 안에 모든 정보를 담음
            - 키와 값으로 하나, 여러 개의 쌍을 만들수 있음 
            - 형식
                객체명 = {
                    키 : 값,
                    키 : 값,
                    .
                    .
                }
        - 객체의 값에 접근하기 
            - 형식
                객체명.키 이름
                객체명["키 이름"]

    7) 배열 
        - 역시 객체임 
        - 형식
            배열명 = [값, 값, ...]
            배열명 = []     // 빈 배열           

3. 자료형 변환 
    1) 자바스크립트 자료형 특징 
        - 느슨한 자료형 체크 
        - 자동 형 변환 
    2) 직접 숫자형으로 변환하기                          
        - Number() 함수 
        - parseInt() 함수, parseFloat() 함수 
    3) 문자열로 변환하기 
        - toString() 함수 
            - null, undefined 자료형을 제외한 나머지 자료형을 
              문자열 데이터로 변환함 
            - 형식
                - 값.toString()
        - String() 함수 
            - null, undefined 자료형까지 모두 포함해서 문자열로 변환함 
    4) 논리형으로 변환하기
        - Boolean() 함수 
            -             true값이 되는 데이터          false값이 되는 데이터                                                     
               ------------------------------------------------------------
                 숫자형         0이 아닌 값                 0
                 문자열      빈문자열이 아닌 모든 문자열    빈 문자열 
                 undefined          -                    undefined
