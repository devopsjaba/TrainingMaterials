1. 로컬에 세팅하기 (기존 방식)
    1) Node.js 설치 
        - 오픈소스 자바스크립트 런타임 환경 
        - 홈페이지에서 LTS (Long Term Support) 버전 설치 
        - 설치 완료 후 명령프롬프트 확인
            - node -v
    2) Visual Studio Code 설치 
    3) 프로젝트 생성하기 
        - npx create-react-app 프로젝트명 (CRA)
    4) 프로젝트 실행 
        - cd reactbasic
        - npm start        

2. 로컬에 세팅하기 (Vite 방식)        
    1) 기존 방식은 세팅에 시간이 오래 걸리고 다소 무겁움 
    2) Vite를 이용하면 빠르게 반영하고 유용함 
        - Frontend 개발 위해 설계된 빌드 도구 
        - React, Vue, Svelte 등 다양한 프레임워크 지원함 
    3) 프로젝트 생성하기
        - npm create vite@latest 프로젝트명  
    4) 프로젝트 실행                       
        - cd 프로젝트명
        - npm install
        - npm run dev

3. Visual Studio Code 설정
    1) Prettier 
    2) ESLint
    3) 기본 formatter 설정 
        - 검색 창에 default formatter 검색 후
            - Prettier - Code formatter 선택 
        - 검색 창에 editor format on save 검색 후 
        - 검색 창에 prettier semi 검색 후   
    4) Tailwind CSS IntelliSense
        - 리액트 컴포넌트의 스타일링 위함 
    5) Simple React Snippets              

4. 프로젝트 구성 
    1) 리액트 : 단독의 애플리케이션 개발 (프런트엔드)
    2) 스프링 부트 : API 서버 개발 (백엔드)

    3) 리액트 프로젝트 생성 (kmall)
        - CRA (create-react-app)을 이용해서 생성 
        - tailwind css react 검색   
            - tailwind css를 이용해서 공통 레이아웃 구성   
        - React-Router 
            - 웹 사이트 개발 중요 단계 
            - 리액트는 SPA로 동작, 여러 컴포넌트로 하나의 페이지에 표시 
            - React-Router로 브러우저 주소창에 따라 다른 화면 표시 
                - 페이지 이동이 가능하도록 컴포넌트들을 구성 
            - React-Router 모듈 추가 
                - npm install react-router-dom     
        - 기본 라우팅 설정 
            - root.js
